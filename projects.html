Proyectos

Detector de lenguaje de señas

Desarrollo de un modelo de aprendizaje profundo basado en MobileNetV2 para reconocer gestos del lenguaje de señas a partir de imágenes, utilizando un dataset de 2700 imágenes clasificadas en diferentes señas.

CodeNova

CodeNova es una web corporativa en PHP con base de datos MySQL. Permite gestionar servicios, portafolio y blog de forma dinámica. Incluye diseño responsive y modales para mostrar proyectos destacados.

Shark Tank EDA

Análisis exploratorio de datos del programa Shark Tank para identificar patrones en las decisiones de inversión. Se evaluaron pitchers aceptados y rechazados, montos ofrecidos por los sharks y sectores de negocio más comunes. El objetivo fue descubrir factores clave para lograr acuerdos exitosos.

Asignación dinámica de buses del Metropolitano

Este proyecto simula y optimiza la asignación de buses en las horas punta del Metropolitano de Lima, utilizando C++ y técnicas algorítmicas. Se basa en la demanda horaria de pasajeros para calcular la cantidad mínima de buses necesarios por franja horaria.

Feane

Este proyecto es un sistema web para una pizzería llamado Feane, que incluye funcionalidades de carrito de compras, registro e inicio de sesión de usuarios. Está desarrollado con HTML, CSS, JavaScript y localStorage para persistencia de datos. Se realizaron pruebas automatizadas con Cypress y reportes con Allure. El flujo de compra es totalmente funcional desde el menú hasta el resumen del carrito.
