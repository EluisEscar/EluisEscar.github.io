Proyectos

Detector de lenguaje de señas

Desarrollo de un modelo de aprendizaje profundo basado en MobileNetV2 para reconocer gestos del lenguaje de señas a partir de imágenes, utilizando un dataset de 2700 imágenes clasificadas en diferentes señas.

CodeNova

CodeNova es una web corporativa en PHP con base de datos MySQL. Permite gestionar servicios, portafolio y blog de forma dinámica. Incluye diseño responsive y modales para mostrar proyectos destacados.

Shark Tank EDA

Análisis exploratorio de datos del programa Shark Tank para identificar patrones en las decisiones de inversión. Se evaluaron pitchers aceptados y rechazados, montos ofrecidos por los sharks y sectores de negocio más comunes. El objetivo fue descubrir factores clave para lograr acuerdos exitosos.
